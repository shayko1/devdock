// And so much more

Everything you need.
Nothing you don't.

☆
Git Worktree Isolation
Every session can run in an isolated git worktree. Main branch stays clean. Multiple Claudes work on the same repo without conflicts.
⚓
Agent Monitoring
Discover and monitor background agents. See schedules, last run results, costs, and trigger manual runs. Real-time status and log viewing.
⚙
MCP Servers & Skills
Configure MCP servers (HTTP and stdio) with a visual editor. Browse and create custom slash commands and reusable skills.
🔍
Smart Autocomplete
Type / for 25+ commands, @ to search files by name, or create custom commands. All in one searchable, keyboard-navigable interface.
♥
Session History & Resume
Browse every past Claude session. Search by title, keywords, or branch. One-click resume with full context restoration.
◍
Project Launchpad
Scan your workspace, tag and manage projects, run them with one click. Port tracking, process management, and log streaming built in.
⚡
AI Coach
Get real-time suggestions as Claude works — tips, code rewrites, follow-up ideas, and commands. Like pair programming with a senior engineer.
📄
File Explorer & Diff
Browse project files, view them with syntax highlighting, and see git diffs in a side panel — all without leaving the terminal view.
📸
Image Support
Drag-drop or paste images directly into the chat. Claude sees them in the conversation. Perfect for UI bugs, screenshots, and design reviews.
